## 🌍 About SafeRide Africa

**SafeRide Africa** is a transportation intelligence platform designed to improve commuter safety through:

- Driver verification
- Vehicle information lookup
- Community incident reporting
- Safety awareness
- Trusted commuter contributions

The platform allows users to make informed decisions before entering a vehicle by providing access to community-driven safety information.

Our mission is simple:

> Create safer journeys by giving communities the power to share verified transport experiences.

---

# ✨ Core Features

## 🔎 Driver & Vehicle Search

Users can search for:

- Drivers
- Vehicles
- License plates

and access important safety information including:

- Driver profile details
- Vehicle information
- Verification status
- Community trust indicators
- Previous verified reports


---

## 🚨 Incident Reporting

Commuters can report safety incidents involving:

- Drivers
- Vehicles
- Transport experiences

Reports include:

- Incident category
- Description
- Location
- Date
- Supporting evidence

All reports contribute towards improving community safety awareness.

---

## 🛡 Community Trust System

SafeRide Africa encourages responsible contributions through:

- Verified reports
- Helpful reviews
- Community feedback
- User reputation

Trust information is designed to reward meaningful contributions and prevent misinformation.

---

## 👤 User Profiles

Users can build their community identity through:

- Profile information
- Contribution history
- Safety achievements
- Trust level

---

## 🌍 African Coverage

SafeRide Africa is built with an Africa-first approach.

The platform supports:

- African countries
- Major cities
- Local transport environments

Helping communities create safer mobility experiences.

# 🚀 Getting Started

## Requirements

Before running the project locally, install:

- Node.js
- npm


Check your versions:

```bash
node -v
npm -v
```

---

# Installation

Clone the repository:

```bash
git clone <repository-url>
```

Move into the project folder:

```bash
cd SafeRide-Africa
```

Install dependencies:

```bash
npm install
```

---

# Environment Setup

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_api_key_here
```

Add your Google Gemini API key.

---

# Run Development Server

Start the application:

```bash
npm run dev
```

The application will run locally:

```
http://localhost:3000
```

# 🔐 Safety Principles

SafeRide Africa follows these principles:

### Privacy First

User information should be protected.

### Responsible Reporting

### Data Integrity

Trust scores should be based on meaningful community contributions.

### Transparency

Users should understand how information is collected and used.
